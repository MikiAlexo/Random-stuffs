GUIDELINES

  • Conditions of use:

    • You are advised to switch off the device when it’s not working to optimize battery life and reduce overheating
    • Always handle the device with care and keep it away from direct sunlight and dust
    • Do not allow the device to be exposed to adverse weather or environmental conditions such as moisture, humidity, rain, dust etc. It is recommended for the device to operate in temperature ranging 0C to 30C.
    • Do not attempt to open, dismantle or fix the device yourself unless you are an expert.
    • Remember to make backup copies or constantly check the condition of the micro SD card

  • Basics

    How to register

    First connect computer to the device using a cable then open the app called biometric attendance then fill the required information on the specified location, then click the “Save” button. After that place finger on the sensor and register then click the reset button on the device.

    How to Attend

    When turning on the device it will automatically start to scan finger and the user can put finger on the sensor and it will save time on the memory.

    How to remove user

    You can remove user from the memory via the app and then going to the delete page and filling the requirements.

    Common errors

    Communication error

    This usually occurs when the finger print sensor is not connected to the main circuit or when the sensor is not working

    Memory card not found

    When this happens the device will automatically start to display time. And this can be solved by inserting a memory card and then pressing the reset button.

    Time limit

    When someone places their finger on the device it will not automatically save time, first it will check if the person is within the time limit given by the owner. If the device displays something like “on time limit” that means that the device is temporally off and will not take attendance.

    How to read files from memory

    Data base

    Every data that was inserted in the app is stored inside the text file data base. It stores data such us Name,Id and salary. DO NOT EDIT THIS FILE because if you do so the entier attendance might stop working properly or entierly.

    Absent

    Absent-"Date" is where absent employees are stored to defferenciate them from the employees that came early and to make it easier to take the absent people name and id.The file will be created daily and the name will change each day in the format "Absent*Date*" and store data.the data of the previous day won't affect the current one which makes it easier to understand.

    *Month**Day*(attendance)

    This is the file where the time in and out of the employee is stored and it is the data base for the identification of the absent employees. Editing this file is not recommended and it could affect the file that identifies the absent employees for the day.
